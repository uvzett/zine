## Zine 是什么？

Zine 这个名字来自英文"Magazine"的后缀，取的就是"自出版"的意思——每个普通人都能出版自己的内容。

它不是那种一打开就密密麻麻工具栏的专业软件，而是走"拖按式"操作路线：手指点一下段落，滑动就能改字体、换颜色、调排版，所见即所得，跟玩游戏差不多直接。

支持平台：**iOS、Android、Web 端**全覆盖，手机码字、电脑排版，三端实时同步，内容永不丢失。

到目前为止，用 Zine 创作的文章数量已经突破了 **4100 万篇**，算是国内图文排版应用里相当老牌的一款。
